Pseudo-element

Ein CSS pseudo-element benutzt man um bestimmte Teile eines Elements anders aussehen zu lassen. Zum Beispiel kann man es so verwenden:

  • Den ersten Buchstaben oder die erste Zeile eines Elements anders aussehen lassen.
  • Dinge vor oder nach einem Element einfügen.

Pseudo-elements haben folgenden Syntax:

selector::pseudo-element {
property: value;
}

Das ::first-line pseudo-element

Mit dem ::first-line pseudo-element kann man das Aussehen der ersten Zeile eines Text-Elements (z.B. p-Element) verändern.

p::first-line {
color: red;
font-variant: small-caps;
}

Das ::first-letter pseudo-element

Mit dem ::first-letter pseudo-element kann man das Aussehen des ersten Buchstaben eines Text-Elements (z.B. p-Element) verändern.

p::first-letter {
color: red;
font-variant: xx-large;
}

Man kann pseudo-elements auch mit CSS-classes kombinieren und man kann mehrere pseudo-elements bei dem selben Element verwenden.

Das ::before pseudo-element

Mit dem ::before pseudo-element kann man vor einem Element Dinge einfügen. Im Beispiel wird vor dem p-Element mit der "test1"-id ein Bild eingefügt.

p#test1::before {
content: url("../../../../images/home-exercises/SuitImageSmall.jpg");
}

Dieses p-Element hat die id "test1".

Das ::after pseudo-element

Das ::after pseudo-element funktioniert nach dem selben Prinzip wie das ::before pseudo-element, nur dass etwas nach dem Element eingefügt wird.

p#test2::after {
content: url("../../../../images/home-exercises/SuitImageSmall.jpg");
}

Dieses p-Element hat die id "test2".

Das ::selection pseudo-element

Mit dem ::selection pseudo-element steuert man den Teil eines Elements an, der vom Benutzer ausgewählt ist.

p#test3::selection {
color: tomato;
background: #f4a261;
}

Wählen Sie in diesem p-Element etwas aus, um das Beispiel zu testen.
