Combinators

Ein CSS-Selector kann mehr als einen einfachen Selector beinhalten. Zwischen den simplen Selectors, kann man einen Combinator einbauen. Davon gibt es vier verschiedene in CSS:

  • Descendant Selector (Leerzeichen)
  • Child Selector (>)
  • Adjacent Sibling Selector (+)
  • General Sibling Selector (~)
Selector Beispiel Erklärung
element element div p Wählt alle p-Elemente aus, die sich in div-Element befinden.
element > element div > p Wählt alle p-Elemente aus, deren parent ein div-Element ist.
element + element div + p Wählt das erste p-Element aus, das direkt nach div-Elementen platziert ist.
element1 ~ element2 p ~ ul Wählt jedes ul-Element aus, vor dem sich ein p-Element befindet.
