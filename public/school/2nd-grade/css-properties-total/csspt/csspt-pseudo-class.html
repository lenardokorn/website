Pseudo-class

Pseudo-classes werden benutzt, um einen bestimmten Zustand eines Elements zu definieren. So kann man:

  • Ein Element verändern wenn sich der Mauszeiger darauf befindet.
  • Besuchte und nicht besuchte Links anders aussehen lassen.
  • Ein Element verändern, wenn es im Fokus ist.

Pseudo-classes haben in CSS folgenden Syntax:

selector:pseudo-class {
property: value;
}

Pseudo-classes bei Links

  • a:link: nicht besuchter Link.
  • a:visited: besuchter Link.
  • a:hover: wenn sich Mauszeiger auf Link befindet.
  • a:active: ausgewählter Link.

Pseudo-classes können auch mit classes kombiniert werden:

a.highlight:hover {
color: red;
}

Link mit "highlight" als class

Dies funktioniert auch bei anderen Elementen wie z.B. divs.

"Tooltip Hover"

Mit folgendem code erscheint ein p-Element, wenn man über ein div-Element hovert.

p {
display: none;
background-color: #f4a261;
padding: 20px;
}

div:hover p {
display: block;
}

Hier die Maus bewegen, um das p-Element anzuzeigen.

Die :first-child pseudo-class

Mit der :first-child pseudo-class findet man Elemente, die die ersten children anderer Elemente sind.

Die :lang pseudo-class

Mit der :lang pseudo-class kann man für bestimmte Sprachen Regeln festlegen.
